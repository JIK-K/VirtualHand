# VirtualHand Physics Simulator

웹캠 하나만으로 가상 3D 공간 내의 다관절 로봇 손을 제어하여 공을 바구니로 옮기는 물리 기반 상호작용 시뮬레이터입니다.

---

## 🛠 사용 기술 및 라이브러리 (Tech Stack & Libraries)

| 분류 | 라이브러리 | 사용 목적 |
| :--- | :--- | :--- |
| **Language** | `Python` 3.10 | 시스템 통합 및 비즈니스 로직 작성 |
| **Physics Engine** | `pybullet` | 3D 물리 공간 구현(중력, 마찰력, 다관절 로봇, 강체 충돌 연산) |
| **ML & Tracking** | `mediapipe` | 웹캠에서 21개 손가락 관절(Landmark)의 3D 좌표 실시간 추출 |
| **Computer Vision** | `opencv-python` | 웹캠 영상 캡처 및 화면 렌더링 |
| **Math & Vector** | `numpy` | 3D 좌표 변환 및 벡터 연산(쿼터니언 변환 등) |

## ✋ 손 캡처 알고리즘 및 데이터셋 (Hand Tracking)

본 프로젝트는 손의 관절을 캡처하기 위해 구글의 **MediaPipe Hands** 사전 학습 모델(Pre-trained Model)을 사용합니다.
- **파이프라인 구조:** 전체 이미지에서 손바닥의 Bounding Box를 먼저 찾는 모델(Palm Detector)과, 찾아낸 손 영역 안에서 21개의 3D 관절 좌표를 추론하는 모델(Hand Landmark Model)의 2단계 구조로 동작합니다.
- **학습 데이터셋:** 구글 연구진이 수동으로 3D 좌표를 어노테이션(Annotation)한 약 30,000장 이상의 실제 손 이미지 데이터와, 다양한 배경에 합성된 3D 렌더링 손 모델 데이터를 기반으로 학습된 모델을 활용했습니다.

## ⚙️ 구현 디테일 (Implementation Details)

### 1. 카메라 좌표를 3D 공간으로 변환 (`utils.py`)
- **거리 왜곡 보정:** 손이 카메라에 가까워지면 크기가 커지는 원근 왜곡을 막기 위해, 손목과 중지 뿌리 사이의 거리를 계산하여 가상 공간 내 로봇 손의 크기가 항상 일정하게 유지되도록 스케일(Scale)을 보정합니다.
- **깊이(Depth) 고정 및 조작성 강화:** 2D 카메라로 앞뒤(Y축) 깊이를 조작하는 어려움을 없애기 위해 로봇 손의 위치를 바구니가 있는 라인(Y=0.5)에 고정했습니다. 또한 좌우(X축) 및 상하(Z축) 좌표의 이동 폭을 2.5배로 늘려 작은 손짓만으로도 편하게 화면 전체를 조작할 수 있도록 구현했습니다.

### 2. 물리 상호작용 제어 (`robot_simulator.py`)
- **로봇 손 구조:** 21개의 구(Sphere)와 20개의 캡슐(Capsule)형 강체를 조합해 손가락 마디를 구성했으며, 손바닥은 실린더(Cylinder)를 배치해 공을 쉽게 떠낼 수 있도록 만들었습니다.
- **물리 튕김(Glitch) 현상 제어:** 손가락 강체가 공과 부딪힐 때 공이 비정상적으로 튕겨나가는 현상을 막기 위해, 각 관절의 제어 힘(`maxForce`)을 크게 낮추어 실제 손처럼 부드럽게(Soft) 움직이도록 제한했습니다. 대신 물체가 미끄러지지 않도록 표면 마찰력(`lateralFriction`)을 높이고 공의 질량(Mass)을 가볍게 조절하여 그립감을 향상시켰습니다.

## 🚀 설치 및 실행 방법

PyBullet 구동 환경을 위해 **Conda 가상환경** 사용을 권장합니다.

### macOS / Linux
```bash
./setup_and_run_conda.sh
```

### Windows
```cmd
setup_and_run_conda.bat
```

## 🎮 조작 방법
1. 프로그램을 실행하면 웹캠 화면과 3D 가상 공간(PyBullet)이 동시에 열립니다.
2. 웹캠 화면 중앙에 손을 펴서 보여주면 가상 공간에 로봇 손이 인식됩니다.
3. 카메라 앞에서 손을 **상/하/좌/우**로 움직여 우측 바구니에 있는 공 위로 위치시킵니다. (앞뒤로 뻗을 필요 없음)
4. 손가락을 굽혀 공을 집거나 손바닥으로 퍼 담은 뒤, 좌측 바구니로 옮겨 떨어뜨립니다.
5. 종료하려면 웹캠 화면(OpenCV 창)이 활성화된 상태에서 `q` 키를 누르세요.
